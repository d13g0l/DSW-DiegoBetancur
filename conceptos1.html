Formas de aplicar CSS


Existen dos formas adicionales de aplicar estilos CSS: en la parte inicial del documento HTML (aplicación de estilos interna o CSS interno) o en un archivo de extensión .css independiente del archivo HTML (aplicación de estilos externa o CSS externo).


CSS INTERNO


Dentro de las etiquetas <head> … </head> incluimos una etiqueta de apertura donde declaramos los estilos <style type=”text/css”>, y a continuación colocaremos la definición de tantos estilos deseemos y cerraremos la declaración con </style>. En HTML 5 no es necesario especificar type =”text/css”.

Al aplicar estilos en línea, tenemos que repetir la aplicación de estilos en cada una de las líneas que queramos modificar y en cada ocasión afectamos únicamente una línea. Con la aplicación de estilos interna nos basta con declarar una vez el estilo y el tipo de elemento al que se aplica, y automáticamente se aplicará a todos los elementos de ese tipo existentes dentro de la página web. Esto permite que en una página web de gran extensión nos ahorremos tener que escribir múltiples veces la definición de estilos (una en cada línea), ya que una sola declaración inicial nos bastará, lo cual es una ventaja bastante evidente.


CSS EXTERNO


Aunque el CSS interno nos permite unificar en una declaración todos los estilos para un archivo html, seguimos teniendo el problema de tener que repetir la definición de estilos en la cabecera de cada uno de los archivos html de nuestro desarrollo web.

Es importante tener en cuenta que si la página contiene pocas líneas en su código quizás sea más sencillo su concepción y esta metodología de trabajo, pero cuando el código empleado tiene cientos o miles de archivos y líneas, se convierte en un problema el manejo y diseño, ya que cada vez que introdujéramos cambios debemos hacerlo en los múltiples archivos de que constara el desarrollo.

Con el fin de optimizar los tiempos de diseño y codificación se ideó la declaración externa de CSS, la cual está basada en declarar los estilos CSS en un archivo independiente que puede servir como referente para dotar de estilos a decenas, cientos o miles de archivos html, que únicamente deberán invocar el nombre de archivo utilizando una sintaxis específica. De este modo un cambio en los estilos habrá que hacerlo únicamente en el archivo de estilos correspondiente, lo cual supone una gran ventaja. Incluso un cambio completo de los estilos de una página web se puede conseguir simplemente sustituyendo el archivo correspondiente.



SELECTORES CSS


Los selectores, son objetos que utilizaremos para seleccionar una serie de etiquetas a los que por medio de la modificación de sus atributos nos permitirán editar el diseño del contenido y dentro de los más empleados tenemos:


  • Clase - Su Sintaxis es: .nombreClase
  • Id - Su Sintaxis es: #nombreId
  • Conjunto de elementos - Su Sintaxis es: h1 , h2 , h3
  • Selectores descendientes - Su Sintaxis es: article p
  • Selectores de hijos directos - Su Sintaxis es: li > ul > li
