CSS3: Estilos internos y Externos, Selectores CSS

Objetivo General

Identificar y conocer los conceptos, herramientas y propiedades para desarrollar sitios web en formato HTML5 y CSS3 y como por medio de los diferentes editores de código y uso de correcto de las etiquetas podemos visualizar el contenido en los diferentes navegadores.

Objetivos Especificos

  • Describir y diseñar cada uno de los procesos o tareas solicitadas en el proyecto.
  • Identificar y conocer el uso de cada una de las etiquetas para una correcta codificación
  • Diseñar una pagina web con diferentes contenidos tales como multimedia, texto, imagenes entre otros.
  • Diseñar un OVI para el curso de Diseños de sitios web.
  • Aplicar conocimiento y lo aprendido sobre el tema seleccionado, CSS3: Estilos internos y Externos,Selectores CSS
